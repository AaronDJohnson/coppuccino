# coppuccino

![Coppuccino Logo](image.png)

Fit distributions with normalizing flows + copulas using JAX. Coppuccino combines the power of normalizing flows with copula models to learn complex probability distributions and generate samples from them.

## What is coppuccino?

Coppuccino is a JAX-based library for fitting and sampling from complex probability distributions using copula flows. It leverages normalizing flows to model marginal distributions and copulas to capture dependencies between variables.

## Installation

### Using pip

```bash
pip install coppuccino
```

### From source

```bash
git clone https://github.com/yourusername/coppuccino.git
cd coppuccino
pip install .
```

### Requirements

- Python >=3.11
- JAX >=0.7.2
- NumPy >=2.3.3
- Pandas >=2.3.3
- PyArrow >=21.0.0
